== Description ==

Hide posts(post, page, post types and attachments) for specifics roles in dashboard.

= Contribute =

You can contribute to the source code in our [GitHub](https://github.com/marcelotorres/hide-posts-for-specific-roles) page.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Settings -> Tools -> Hide posts
